## Key Features

- **Auto-Discovery:** Automatically finds compatible RV-C devices and creates Home Assistant entities.
- **Bidirectional Communication:** Read real-time status from your RV and send commands back to control devices.
- **Comprehensive Device Support:** Works with a wide range of standard RV-C devices.
- **Reliable & Offline-Capable:** Runs entirely inside your RV.
- **DIY Friendly:** Detailed guides for building the hardware.

## How It Works

1. **CAN Bridge Hardware:** Raspberry Pi with CAN HAT connects to RV-C network.
2. **CAN to MQTT Software:** Reads messages and publishes to MQTT.
3. **Node-RED Processing:** Decodes data for Home Assistant.
4. **Home Assistant Integration:** Auto-configures entities via MQTT Discovery.
